# Soukromi a Metadata (CZ)

E2EE chrani obsah zprav, ale provozni metadata existuji.

## Typicka metadata

- identifikatory odesilatel/prijemce,
- casove znacky,
- stav doruceni/precteni,
- session/device metadata.

## Proc existuji

Bez metadat nelze delat routing, synchronizaci a multi-device provoz.

## Privacy strategie

- minimalizace metadat,
- zadny plaintext na backendu,
- omezeni logovani citlivych dat,
- jasna retention politika.
