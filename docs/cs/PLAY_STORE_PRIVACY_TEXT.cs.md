# Play Store Privacy Texty (CZ)

Tento soubor je sablona textu pro Play Console (Data Safety + App Content).
Pred publikaci zkontroluj pravni presnost.

## Kratky popis

GitMit pouziva end-to-end sifrovani obsahu zprav. Backend zpracovava provozni metadata nutna pro doruceni a synchronizaci.

## Security practices

- data in transit jsou sifrovana,
- obsah zprav je sifrovany v E2EE flow,
- backend pouziva authorization controls,
- bezpecnostni problem lze hlasit podle SECURITY.md.

## Poznamka

Neslibuj "zero metadata" ani "perfect security", pokud to implementace realne nedava.
