# Backend Security Checklist (CZ)

- [ ] Vynutit auth pro vsechny soukrome operace.
- [ ] Deny-by-default authorization rules.
- [ ] Server-side validace vstupu a schema kontrola.
- [ ] Limity velikosti payloadu.
- [ ] Rate limiting a anti-abuse ochrany.
- [ ] Zadny plaintext citlivych dat v logu.
- [ ] Monitoring auth failure a anomalii.
- [ ] Rotace secretu po incidentu.
